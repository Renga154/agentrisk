# AgentRisk v0.1.0

Initial public release of AgentRisk: a zero-execution preflight scanner for untrusted AI-agent and MCP artifacts.

## Highlights

- Scan local folders, GitHub repositories, npm package tarballs, and local/remote archives before an AI coding agent opens or runs them.
- Detect risky MCP launch commands, privileged containers, shell wrappers, unpinned launchers, secret env forwarding, prompt-injection-style repo instructions, approval bypasses, and install-time remote execution.
- Emit Terminal, JSON, Markdown, and SARIF reports for local review and GitHub code scanning.
- Ignore target-provided AgentRisk config for remote/npm/archive scans unless `--config` is explicitly provided.
- Fail closed when parsing or reading important files is incomplete.
- Include a checked-in corpus currently reporting 3/3 benign cases clean and 5/5 malicious cases detected.

## Try It

```bash
npm exec --yes github:Renga154/agentrisk -- scan .
npm exec --yes github:Renga154/agentrisk -- scan https://github.com/owner/suspicious-agent-repo
npm exec --yes github:Renga154/agentrisk -- scan npm:some-mcp-server@1.2.3
```

After npm registry publication, the same commands can be shortened to:

```bash
npx agentrisk scan .
```

## GitHub Action

```yaml
- uses: Renga154/agentrisk@v0.1.0
  with:
    format: sarif
    output: agentrisk.sarif
```

## Security Model

AgentRisk never connects to MCP servers, never runs package scripts, never installs target dependencies, and never sends workspace contents to an LLM or remote model provider.
